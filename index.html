GERADOR DE PORTARIAS

1. Dados do Documento

Selecione o comando responsável.
A unidade é obrigatória.
O tipo de portaria é obrigatório.
O número da portaria é obrigatório.
Data inválida.

2. Dados do Militar Encarregado

Obrigatorio selecionar posto.
A matrícula deve ter 8 dígitos (prenchidos: 0/8).
Nome do encarregado é obrigatório.

3. Fundamentação e Histórico

Prenchimento obrigatório.
Prazo inválido (apenas números positivos).
Cidade é obrigatória para a assinatura.
A descrição dos fatos não pode ser vazia.

4. Emissão e Autoria

Obrigatorio selecionar posto do Cmt.
O nome do Comandante é obrigatório.
